# 🚀 AI Engineering Portfolio & Learning Projects

ברוכים הבאים למאגר הפרויקטים שלי בתחום ה-AI Engineering! 
מאגר זה מרכז יישומים וכלים שפיתחתי, המדגימים שילוב של מודלי שפה גדולים (LLMs), סוכנים חכמים (AI Agents), וממשקי משתמש (API ו-Web UI) לפתרון בעיות מעשיות.

לכל פרויקט יש תיקייה ייעודית הכוללת את קוד המקור, קובץ התקנה (`requirements` / `pyproject.toml`), וקובץ `README.md` מפורט עם הוראות הרצה.

## 🛠️ ארכיטקטורה וטכנולוגיות (Tech Stack)

הפרויקטים במאגר זה מדגימים שימוש נרחב בכלים הבאים:
* **AI & LLMs:** שימוש ב-API של חברות כמו Groq ו-xAI, בניית System Prompts מורכבים, ויצירת סוכנים (Function Calling / Tool Use).
* **Backend Development:** בניית שרתי REST מהירים ועצמאיים בעזרת **FastAPI**.
* **Frontend / UI:** פיתוח ממשקי משתמש מבוססי צ'אט ואינטראקציה מהירה בעזרת **Gradio**.
* **ניהול סביבות:** עבודה עם `uv`, `venv`, וניהול משתני סביבה בעזרת `python-dotenv`.
