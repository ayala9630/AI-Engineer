# איך לתרום לפרויקט [שם הפרויקט]

אנחנו שמחים שאתה מעוניין לתרום! כל תרומה עוזרת ומוערכת.

## 🐞 דיווח על באגים

*   ודא שהבאג לא דווח כבר על ידי חיפוש ב-Issues.
*   פתח Issue חדש וספק תיאור מפורט של הבעיה, כולל צעדים לשחזור, גרסת התוכנה ומערכת ההפעלה.

## 💡 הגשת הצעות לשיפורים

*   פתח Issue חדש ותאר את ההצעה שלך. הסבר מדוע השיפור הזה נחוץ ומה היתרונות שלו.

## 🛠️ תהליך הגשת Pull Request (PR)

1.  בצע Fork למאגר (Repository).
2.  צור ענף (Branch) חדש עבור השינוי שלך (`git checkout -b feature/amazing-feature`).
3.  בצע את השינויים שלך ובצע Commit (`git commit -m 'Add some amazing feature'`).
4.  דחוף את השינויים לענף שלך (`git push origin feature/amazing-feature`).
5.  פתח Pull Request למאגר המקורי.

## ✅ סטנדרטים של קוד

*   ודא שהקוד שלך עומד בסטנדרטים של הפרויקט (לדוגמה, השתמש ב-Linter שהוגדר).
*   כתוב תיעוד רלוונטי לקוד החדש.
*   הוסף בדיקות יחידה (Unit Tests) במידת האפשר.
